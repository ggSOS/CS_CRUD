# CS_CRUD
Exercício de BackEnd para C#

## Requisitos
- Instalar pacotes:
  - Microsoft.Extensions.Configuration
  - MySqlConnector
- Servidor MySQL em funcionamento
- Definir as seguintes variáveis de ambiente(referentes ao seu banco de dados) dentro do modelo a seguir:
  - DbPort
  - DbName
  - DbUser
  - DbPassword
  ```bash
  dotnet user-secrets set "DbUser" "root"
  ```
